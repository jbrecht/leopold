A fireworks show featuring photos of Leopold!

## Live Site

[**Click here to watch the show!**](https://jbrecht.github.io/leopold/)

## Features

- **Custom Fireworks**: Each explosion releases particles made of photos.
- **Static Hosting**: Refactored to run entirely on GitHub Pages.
- **Easy Updates**: Just add photos to the `photos` folder and update `photos.json`.

## Local Development

To run the site locally:

```bash
node server.js
```

Then visit `http://localhost:8001`.
